== Description ==

Right Bottom Nav 1.0版本1

Plugin URI: http://zhengxianjun.sinaapp.com/plugins-wordpress-right-bottom-nav/
Description: 右下角折叠菜单, 由于WordPress基本都引入了JQuery,所以这里不再添加JQuery源文件,请确保引入了JQuery文件, right_bottom_nav的脚本在页面底部. 支持IE7.0+,Chrome;FF未测试. <code>Developed by 郑显军</code>.
Version: 1.0
Author: 郑显军
Author URI: http://zhengxianjun.sinaapp.com/

== Installation ==

1. 上传到wp-contents/plugins/文件夹下
2. 在后台启动'Right Bottom Nav',由于没有提供定制功能,所以没有添加到'设置栏'

== Frequently Asked Questions ==
功能
	1 页面的滑动
	2 文章列表
		显示热点文章(图片式,数量3),最新文章(列表式,数量10),随机文章(列表式,数量10)
	3 天气预报
		调用Google天气预报api实现,谷歌有时候不稳定,所以访问出错时调用中国天气网的天气预报

使用
	小巧,自动收回,无法定制,不影响数据库
	
样式表
	使用csssprites.com生成sprite,所以如果需要修改,需要修改images文件夹下对应的小图片,重新生成并替换即可.
